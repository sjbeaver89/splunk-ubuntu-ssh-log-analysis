
# Splunk SSH Log Analysis on Ubuntu VM

## Project Overview
This project demonstrates deploying **Splunk Enterprise** on an **Ubuntu virtual machine (VM)**, ingesting Linux authentication logs (`auth.log`), and building a dashboard to visualize SSH login activity.  
The goal is to showcase **log analysis, field extraction, and dashboard creation**, suitable for a cybersecurity or data analytics portfolio.

---

## Objectives
- Install and run Splunk Enterprise on Ubuntu
- Ingest Linux authentication logs
- Troubleshoot common Linux and Splunk issues
- Extract fields from raw logs using regex
- Build dashboards to visualize SSH login attempts
- Simulate login activity to test dashboards

---

## Environment
- **Host OS:** Windows / macOS / Linux
- **Hypervisor:** Oracle VM / VirtualBox
- **Guest OS:** Ubuntu 24.04 Desktop/Server
- **SIEM:** Splunk Enterprise (Free)
- **Browser:** Firefox inside VM
